
### - RoutingTable (ConfigSource)

> Retrieves the routing table from the device
> 
> User needs to have enable access (#), if it doesn't login directly into enable mode please define the property 'ssh.enable.pass' to further elevate it.

> Supported devices:
> -
> - Cisco IOS devices
> - Cisco NX-OS
> - Cisco ASA
> - Palo Alto Firewall(s)
